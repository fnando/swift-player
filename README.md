# swift-player

Control your media player using MediaRemote.

```console
$ player -h
OVERVIEW: Control the media player

USAGE: player <subcommand>

OPTIONS:
  -h, --help              Show help information.

SUBCOMMANDS:
  track                   Get info about the current playing track
  play                    Play track
  pause                   Pause track
  stop                    Stop track
  next                    Play next track
  previous                Play previous track
  play-pause              Toggle play/pause
```
